Command line tool to automatic copy and rename a folder of js classes


## Installation
```{r, engine='bash', global_install}
npm install @marketto/class-mitosis -g
```


## Getting Started
```{r, engine='bash', run}
mitosis -s source-dir -d destination-dir
```
